# 剧情配置工具 — 快速启动指南

## 一、首次启动

```bash
cd e:\work\剧情配置工具
npm install           # 安装依赖（首次需5-10分钟，包含Electron下载）
npm run electron:dev  # 启动Electron开发模式
```

## 二、使用流程

```
1. 启动工具 → 看到顶部工具栏
2. 点击「📂 加载文件夹」→ 选择包含 StoryFrame.xlsx / StoryGroup.xlsx / StoryBehavior.xlsx 的文件夹
   或者直接拖入文件
3. 看到剧情流程图（每个节点 = 一帧，连线 = 剧情跳转，橙色虚线 = 分支选项）
4. 点击任意节点 → 右侧编辑面板出现
5. 修改台词/角色/背景/行为 → 流程图实时更新
6. 点击「📤 导出Excel」→ 保存为xlsx
```

1. **零学习成本配表** — 拖入Excel → 自动可视化
